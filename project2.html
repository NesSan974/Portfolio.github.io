Serious Game

BUT informatique 2eme année

Dans le cadre de notre cursus, nous avons réalisé un mini projet intitulé "L'école des Loustics". Ce projet vise à développer une application éducative pour enfants.La qualité étant primordiale, nous nous sommes concentrés sur des fonctionnalités essentielles et robustes.

Le projet inclus les fonctionnalités suivants :

  • Gestion des comptes pour enfants : Une interface simple pour la création et la gestion de comptes d’utilisateurs sans mot de passe.
  • Jeu de mathématiques : Un jeu éducatif pour apprendre et pratiquer les tables de multiplication.
  • Jeu de culture générale : Un quiz interactif pour renforcer les connaissances générales des enfants.

les objectifs du projet sont :

  • Gestion de la Pile d'Activités : Navigation entre les jeux et l'interface de gestion de comptes.
  • Stockage en Base de Données : Enregistrement et récupération des scores et des informations utilisateur.
  • Cycle de Vie : Gestion des états de l'application pour garantir une expérience utilisateur fluide et sans interruption.

Fonctionnalités de l'Application

Gestion des Comptes

  • Interface simple permettant aux enfants de créer et de gérer leurs comptes sans mot de passe
  • Stockage des informations de compte dans une base de données locale

Jeu de Mathématiques

  • Un jeu interactif pour apprendre les tables de multiplication

Jeu de Culture Générale

  • Un quiz sur des sujets variés pour tester et améliorer les connaissances générales des enfants
  • Questions à choix multiples avec tiré au hasard depuis une base de donnée local
